Allgemeine Informationen

Botsupport

World of PADMAN bietet volle Bots-Unterstützung für alle Spielmodi! Offline-Spieler werden sich freuen, Spray Your Color, Big Balloon, Last Pad Standing, Capture the Lolly und viele mehr mit Bots spielen zu können. Die Unterstützung für Bots funktioniert recht gut und auch die Maps wurden ein wenig für Bots optimiert.

Gegenstände abwerfen

Ihr habt viele Farbpatronen gesammelt und glaubt, dass ihr es nicht mehr bis zum Teleporter schafft? Ihr wollt eurem Teamkollegen eine Farbpatrone geben, damit er mit SprayGod oder SprayKiller zusätzliche Punkte für das Team sammeln kann? Für solche Fälle könnt ihr im Menü unter Setup -> Controls -> Move eine Taste zum Abwerfen definieren. Voreingestellt sind hier 'F' und 'Backspace'. Dieses Feature funktioniert nicht nur in der Teamversion von Spray Your Color, sondern auch in Capture The Lolly und One Lolly Capture. Hier kann man den Lutscher an andere Spieler weitergeben. Bei Catch The KillerDuck kann man so die Ente schnell wieder loswerden, wenn sie einem zu viel wird.

Glow-Skins

Bei ernsthaften Turnieren ist es wichtig, den Gegner schnell zu erkennen. Aus diesem Grund bietet WoP die Möglichkeit, im Menü unter Setup -> Options -> Game sogenannte Glow-Skins zu aktivieren. Wenn ihr dies tut, werden die Modelle eurer Gegner mit einem leuchtenden Skin in einer Farbe eurer Wahl versehen, so dass sie sich deutlich von der Spielwelt abheben. Besonders für Clans und actionreiche 1on1-Kämpfe ist diese Option sicherlich ein Segen, für das grafische Gesamtkonzept allerdings nicht ;) In den Teamspielmodi sind die Farben immer auf rot und blau festgelegt.

Voice-Chat

Vorbei sind die Zeiten, in denen man die Tastatur quälen musste, denn World of PADMAN verfügt über ein Voice-Chat-System. Dieses muss im Menü unter Setup -> System -> Net/VoIP aktiviert werden. Ihr habt dann die Wahl, ob eure Stimme automatisch erkannt und übertragen werden soll, oder ob ihr lieber wie bei einem Walkie-Talkie eine Taste drücken wollt, um zu sprechen (push to talk). Entscheidet ihr euch für die 'push to talk'-Variante, könnt ihr im Menü unter Setup -> Controls -> Chat noch eine Taste dafür festlegen. Voreingestellt ist hier 'R'.
Im Spiel selbst könnt ihr im Ingame-Menü über 'Escape'auch die Lautstärke der einzelnen Spieler regeln oder sie stumm schalten. Wer gerne Mumble als externes Voice-Chat-Programm nutzen möchte, kann im Menü auch den Mumble-Support auswählen.

Warmup

Der Befehl g_WarmupReady kann verwendet werden, damit ein Spiel nicht sofort nach dem Laden startet, sondern erst dann, wenn sich eine bestimmte Anzahl von Spielern sich mit dem Befehl /ready als spielbereit gemeldet hat. Der Serveradministrator kann in der Serverkonfiguration die Variable g_WarmupReady auf einen Wert zwischen 0 und 1 setzen, wobei g_doWarmup ebenfalls auf 1 stehen muss. Wenn g_WarmupReady auf 0 gesetzt ist, bedeutet dies, dass das Spiel sofort gestartet wird. Bei 1 müssen 100% der anwesenden Spieler /ready in die Konsole eingeben. Zusätzlich können alle anderen Dezimalzahlen zwischen 0 und 1 eingegeben werden, um einen bestimmten Prozentsatz an spielbereiten Spielern zu erzwingen.

Beispiele:

set g_doWarmup 1
set g_WarmupReady 0.5 -> 50% müssen spielbereit sein

oder

set g_doWarmup 1
set g_WarmupReady 0.8 -> 80% müssen spielbereit sein

Dieser Befehl wird besonders für Last Pad Standing und generell für teamorientierte Spielmodi empfohlen.

Bugs

World of PADMAN wird seit über 20 Jahren entwickelt, was aber nicht bedeutet, dass es fehlerfrei ist. Wie jeder Hersteller, der etwas auf sich hält, haben wir absichtlich ein paar Bugs eingebaut, damit ihr etwas zum Suchen und Aufregen habt. Wenn ihr Bugs findet, schickt uns eine E-Mail, damit wir mit einem Patch prahlen können. ;) Besser ist es, den Fehler direkt auf Github zu melden, indem ihr dort ein Ticket erstellt.
